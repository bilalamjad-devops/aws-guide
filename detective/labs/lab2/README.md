## 🪜 Steps

### Step 1: Generate activity

* Run multiple commands:

  * Create S3 bucket
  * Modify IAM roles

---

### Step 2: Trigger detection

* GuardDuty may detect unusual activity

---

### Step 3: Investigate in Detective

* Open Detective
* Search IAM user

---

### Step 4: Analyze

Check:

* API calls timeline
* Source IP
* Geo location
* Behavior graph

* From where
* When
