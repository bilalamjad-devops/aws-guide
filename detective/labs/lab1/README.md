

# 🔬 Lab 1: Enable AWS Detective

📁 `lab1-enable-detective/README.md`

## 🎯 Objective

Enable Detective and connect it with GuardDuty.

---

## 🌍 Real-World Scenario

A company wants:

> “Central investigation system for all security findings”

---

## 🪜 Steps

### Step 1: Enable GuardDuty

* Go to GuardDuty
* Click **Enable**

---

### Step 2: Enable Detective

* Go to Detective
* Click **Get Started**
* Enable it

---

### Step 3: Link Data Sources

* CloudTrail logs (auto used)
* VPC Flow Logs
* GuardDuty findings

---

## ✅ Result

Detective starts collecting and analyzing data.

---

## 🧠 Real Meaning

> “We now have visibility to investigate incidents”
